# 日本株・米国株 ランキング取得スキル

## 概要

日本株（かぶたん）と米国株（Yahoo!ファイナンス）から市場ランキングデータを取得し、
Excelファイルとしてデスクトップに保存する。
チームエージェント（3人構成）を使用して日米のデータを並列取得する。

## 対象データ

### 日本株（かぶたん）
- **市場**: 東証全体・プライム・スタンダード・グロース（4市場）
- **ランキング**: 値上がり率・値下がり率・出来高・売買代金（4種）
- **件数**: 各ランキングETF除外で100銘柄
- **ETF除外**: P/S/G選択時は自動除外、東証全体は後処理フィルタ

### 米国株（Yahoo!ファイナンス）
- **市場**: 全市場・NYSE・NASDAQ（3市場）
- **ランキング**: 値上がり率・値下がり率・出来高・売買代金（4種）
- **件数**: 各ランキングETF除外で100銘柄
- **ETF除外**: 後処理フィルタ

## CRITICAL: 実行タイミング

市場終了後に実行すること。
- 日本株: 15:30 JST 以降
- 米国株: 翌朝 6:00 JST 以降（サマータイム時 5:00）

かぶたんはWebFetch（未ログイン）でアクセスする。
市場終了後であれば確定値が取得でき、プレミアムログイン不要。

## ワークフロー

### Step 1: チーム作成・タスク割り当て

TeamCreate でチームを作成し、JP担当・US担当を Task で起動する。

```
チーム名: stock-ranking
構成:
  リーダー（自身）: 全体進行管理・Excel生成・保存
  JP担当: かぶたんから日本株ランキング取得
  US担当: Yahoo!ファイナンスから米国株ランキング取得
```

**CRITICAL**: JP担当・US担当は同時起動し、並列でデータ取得を行うこと。

### Step 2: データ取得（並列実行）

JP担当・US担当にデータ取得を指示する。
各担当の詳細な取得手順は `references/03-team-instructions.md` を参照。

**JP担当の作業:**
- かぶたんから4市場 × 4ランキング = 16パターンのデータを取得
- 各ランキング50件 × 2ページで100銘柄取得
- 東証全体のデータからETF/REIT/ETN/インフラファンドを除外
- URL定義は `references/01-url-definitions.md` を参照

**US担当の作業:**
- Yahoo!ファイナンスから3市場 × 4ランキング = 12パターンのデータを取得
- 各ランキング50件 × 2ページで100銘柄取得
- ETF銘柄を後処理で除外
- URL定義は `references/01-url-definitions.md` を参照

### Step 3: データ受信・統合

各担当から取得結果を受信し、データの整合性を確認する。

確認項目:
- 各ランキングで100銘柄（またはそれに近い数）取得できているか
- ETFが除外されているか
- データ項目が揃っているか

### Step 4: Excelファイル生成

openpyxl を使用してPythonスクリプトでExcelファイルを生成する。
シート構成・スタイル定義は `references/02-excel-templates.md` を参照。

**CRITICAL**: 必ず Write ツールでPythonスクリプトを保存し、Bash で実行すること。

**出力ファイル:**
```
デスクトップ/
├── JP_株式ランキング_YYYY-MM-DD.xlsx   （日本株: 16シート）
└── US_株式ランキング_YYYY-MM-DD.xlsx   （米国株: 12シート）
```

**ファイル名ルール:**
- 日付は実行日（YYYY-MM-DD形式）
- 日本株: `JP_株式ランキング_2026-03-09.xlsx`
- 米国株: `US_株式ランキング_2026-03-09.xlsx`

### Step 5: 保存・完了報告

- デスクトップにExcelファイルを保存
- チームメイトをシャットダウン
- 取得結果のサマリーをユーザーに報告

報告内容:
- 取得日時
- 各ランキングの取得銘柄数
- 保存先ファイルパス
- エラーがあった場合はその内容

## エラーハンドリング

### WebFetchでデータ取得できない場合
```
サイトからデータを取得できませんでした。
考えられる原因:
・サイトのメンテナンス中
・ネットワーク接続の問題
・サイト構造の変更

再実行するか、時間をおいて再度お試しください。
```

### 市場が開場中の場合
```
現在、市場が開場中のため確定データが取得できません。
・日本株: 15:30以降に実行してください
・米国株: 翌朝6:00以降に実行してください

それでも実行しますか？（リアルタイムではなく15分遅延データになります）
```

### ETFフィルタで想定外の銘柄が混入した場合
- 市場コード（東P/東S/東G以外）を持つ銘柄を除外
- 銘柄名に「ETF」「ETN」「REIT」を含むものを除外
- 除外した銘柄数をログに記録

### Excelファイルが開かれている場合
```
ファイルが他のアプリケーションで開かれているため保存できません。
Excelファイルを閉じてから「続けて」と入力してください。
```
