# Teamagent プロジェクト

## 概要
Claude Code Agent Teams を使った並列開発プロジェクト。

## チームエージェント設定
- Agent Teams 機能が有効化済み (`CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`)
- 表示モード: `in-process`（Windows環境のため）
- tmux-windows もインストール済み

## 使い方

### チームを起動する
Claude Code を起動して、自然言語でチーム作成を依頼する：

```
チームを作成して、以下の役割で並列作業してください：
- リサーチャー: 技術調査担当
- 実装者: コーディング担当
- レビュアー: コードレビュー担当
```

### チームメイトの操作
- `Shift+Down`: チームメイト間を移動
- `Enter`: チームメイトのセッションを表示
- `Escape`: チームメイトの現在のターンを中断
- `Ctrl+T`: タスクリストの表示/非表示

## 注意事項
- Windows環境ではスプリットペインモード（tmux分割表示）は制限あり
- in-processモードを推奨
- チームメイトは3〜5人が最適
- 作業完了後は必ずリーダーにチームクリーンアップを依頼する
